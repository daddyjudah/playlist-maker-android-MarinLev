# playlist-maker-android-MarinLev

Учебное приложение для поиска музыки.

### Спринт 4
- Логика очистки поиска и системные переходы.
- Единая навигация.

### Спринт 3
- Создание главного экрана.
- В обработчиках нажатия на соответствующие кнопки нет Toast.
- Базовая вёрстка экранов настройки и поиск.

### Функционал
- Поиск треков
- Настройки 
- Медиатека

### Стек
- Kotlin, Jetpack Compose
- Navigation Compose
- SDK 34

### Запуск
1. Склонировать проект.
2. Открыть в Android Studio.
3. Запустить на эмуляторе или устройстве.
