# playlist-maker-android-MarinLev

Учебное приложение для поиска музыки.

### Спринт 6: Связывание архитектуры с интерфейсом
- Реализована 'SearchViewModel' с управлением состояниями экрана.
- Внедрен 'SearchScreen'.
- Реализован ввод поискового запроса с ручным запуском поиска.
- Настроен 'TrackListItem' для отображения результатов поиска.

### Спринт 5: Внедрение Clean Architecture
- Разделение проекта на слои Data, Domain и UI.
- Реализация эмулятора сервера для имитации сетевых запросов.
- Создание репозитория и сетевого клиента.

### Спринт 4
- Логика очистки поиска и системные переходы.
- Единая навигация.

### Функционал
- Поиск треков.
- Настройки приложения.

### Стек
- Kotlin, Jetpack Compose, Flow.
- Clean Architecture, MVVM.
- Navigation Compose.
